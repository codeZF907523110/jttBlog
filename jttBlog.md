# 贾甜甜的博客
## 技术栈
- vitepress + ts
- git
- node版本20+
- eslint+prettier，要注意适用的版本
- eslint和prettier的配置不能有冲突，且prettier要装prettier-error的插件
- gitcommit时要校验，可以使用husky、commitlint或lint-staged等来实现
- 创建完之后尝试启动，看看控制台是否有报错，有报错的话请解决
- 包版本管理用yarn
## 博客主体
- 整体风格：二次元动漫风格，可爱风
- 博客主要是用来展示作者的作品集，作者是一名优秀的视觉设计师
- 整体做成响应式布局，适配不同屏幕尺寸
- 移动端兼容
- 增加飘桃花效果（所有页面都要有这个效果，全屏，桃花稍微大一点，要覆盖到其它元素上）
### 首页
- 导航栏：
  - 菜单前面增加图标，图标要和菜单文字对齐，二级菜单也要加图标
  - 菜单下拉框样式做成透明+磨砂效果，下拉框的文字要居中
  - 选中之后菜单要有反馈，和鼠标移入的效果一致
  - 移动端导航栏展开会加一个背景颜色
  - 菜单内容
    - 首页
    - 作品集
    - 相册
    - 关于我
    - 更多
      - 友链
      - 分享
- 首页第一屏时banner图，banner图正中间有三个内容：
  - 英文的欢迎来到JTT的博客（只有这儿的英文）
  - 博客的标题：贾甜甜的博客
  - 博客描述：这是一个展示作者作品集的博客，作者是一名优秀的视觉设计师
    - 博客描述要做成打字机的效果，每个字符之间有间隔，像打字机一样，动画循环播放
- 首页下面的结构
  - 左侧是导航卡片：
    - 导航卡片做成吸顶效果，当滚动滚动到导航卡片的位置时，导航卡片会固定在顶部，不会滚动
    - 第一个卡片：卡片里面包含
      - 头像、姓名
      - 文章数量（文字前面加图标）
      - 作品集数量（文字前面加图标）
      - 分类数量（文字前面加图标）
    - 第二个卡片：卡片里面包含分类列表
      - 每个分类列表的tag都是随机的颜色，但是颜色都要清新一点，tag有黑色的边框，圆角的矩形（圆角小一些），分类可以多一些
      - 每个tag会有一个动画效果，tag会一直缓慢的放大缩小（只保留这一个动画效果即可）
      - 鼠标放置到某一个tag之后这个tag高亮且整个球停止旋转
      - 点击某一个tag之后，会跳转到该分类的文章列表页
  - 右侧有多块区域：
    - 每个区域顶部都有一个间隔块
      - 间隔快是一个div，左侧是区域标题，右侧展示一个更多按钮，点击后会跳转到该区域的列表页
    - 第一个区域：作品集展示卡片列表
      - 每个卡片里面包含作品集的图片、标题、描述、分类、创建时间
      - 每个卡片之间有间隔，像卡片一样，每个卡片都有一个阴影效果
      - 动画：鼠标悬停在卡片上时，卡片会有一个放大效果，同时卡片的阴影也会变大
      - 点击卡片之后，会跳转到该作品集的详情页
    - 第二个区域：相册集
      - 每个相册集里面包含相册的图片、标题、描述
      - 每个相册集之间有间隔，像卡片一样，每个相册集都有一个阴影效果
      - 动画：鼠标悬停在相册集上时，相册集会有一个放大效果，同时相册集的阴影也会变大
      - 点击相册集之后，会跳转到该相册集的详情页
- 作品集列表页面
  - 点击菜单栏的作品集会跳转到作品集列表页面
  - 最上面是一个banner图，高度为页面的三分之一左右，宽度是页面的100%，banner图可以参考首页
  - 这儿是标题+描述
  - 下面增加一个分类，根据分类可以进行筛选
  - 首页的分类tag点进来的时候默认自动分类
  - 最多展示三列
  - 作品集列表页面展示所有作品集，每个作品集展示卡片列表
    - 每个卡片里面包含作品集的图片、标题、描述、分类、创建时间
    - 每个卡片之间有间隔，像卡片一样，每个卡片都有一个阴影效果
    - 动画：鼠标悬停在卡片上时，卡片会有一个放大效果，同时卡片的阴影也会变大
    - 点击卡片之后，会跳转到该作品集的页面
  - 作品集详情页
    - 新建多个临时md文件，每个文件对应一个作品集，文件名就是作品集的标题
    - 每个文件里面包含作品集的标题、描述、图片、分类、创建时间
    - 作品集列表展示真实的作品
- 相册
  - 点击菜单栏的相册会跳转到相册列表页面
  - 最上面是一个banner图，高度为页面的三分之一左右，宽度是页面的100%，banner图可以参考首页
  - 这儿是标题+描述
  - 下面增加一个分类，根据分类可以进行筛选
  - 最多展示三列
  - 相册列表页面展示所有相册，每个相册展示卡片列表
    - 每个卡片里面包含相册的图片、标题、描述、分类、创建时间
    - 每个卡片之间有间隔，像卡片一样，每个卡片都有一个阴影效果
    - 动画：鼠标悬停在卡片上时，相册会有一个放大效果，同时相册的阴影也会变大
    - 点击相册卡片之后，会跳转到该相册的详情页
    - 相册的卡片做的不要和作品集一样，可以稍微大一些
  - 相册详情页
    - 新建多个临时md文件，每个文件对应一个相册，文件名就是相册的标题
    - 每个文件里面包含相册的标题、描述、图片、分类、创建时间
    - 相册列表展示真实的作品相册
